### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 流浪猫狗
5. 流浪猫狗知识
6. 团队信息
7. 活动信息
8. 宠物资讯
9. 后台管理
10. 个人中心
11. 我的收藏


**管理员功能：**
1. 登录
2. 系统首页（宠物类别统计、领养状态统计、每日申请人数统计）
3. 个人中心（修改密码、个人信息）
4. 用户管理
5. 宠物分类管理
6. 知识类型管理
7. 流浪猫狗管理
8. 领养信息管理
9. 流浪猫狗知识管理
10. 团队信息管理
11. 活动分类管理
12. 活动信息管理
13. 志愿申请管理
14. 系统管理(关于我们、系统简介、轮播图管理、宠物资讯)
